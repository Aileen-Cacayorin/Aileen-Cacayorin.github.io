RSVP

An event planning tool.

RSVP is a planning tool for event based travel (weddings, family reunions, etc). An event host can input important event details such as dates and location and then invite guests . Invited guests can then view those details, view the guest list, see what hotels other guests are staying at, and book hotels based on where other guests are staying. The app was conceived, designed, coded, and deployed over the span of one weekend and was a Finalist at the Expedia Seattle Hackathon.

Built with Ruby on Rails, Google Maps Javascript API and Expedia API.
