RSVP

Built with Ruby on Rails, Google Maps Javascript API, and Expedia API.

RSVP is a planning tool for event based travel (weddings, family reunions, etc). An event host can input important event details such as dates and location and then invite guests . Invited guests can then view those details, view the guest list, see what hotels other guests are staying at, and book hotels based on where other guests are staying. The app was conceived, designed, coded, and deployed over the span of one weekend and was a Finalist at the Expedia Seattle Hackathon.

My Piggy Bank

Built with Ruby on Rails, Google Charts, and Chartkick.

My Piggy Bank is an online tool to help parents teach their kids about money. Parents create a family bank, add accounts for their children, and act as the banker. Parents can login, view their children's savings, account history, and approve transactions for allowances, chores, toys etc. Children can login to their account and view their curent savings. If is a child has actual cash to deposit or needs cash to spend, they can make a transaction request from their account. A parent can then log in, approve the request, exchange the money, and the new transacton is recorded.

GYR (Green|Yellow|Red)

Built with Ruby on Rails, PostgreSQL, and MailGun.

A web dashboard for teachers to manage student records while keeping parents fully informed. Teachers can track behavior, make individual comments on students, mark absences, detail the day's lessons, detail and attached the evening's homework, and finally email parents all that information from one screen.The app was conceived, designed, and built in under 10 hours, and won a Portland's HackPDX Winter 2015 Hackathon. It is built with Ruby on Rails, PostgreSQL, Action Mailer (with MailGun), and wire-framed with Bootstrap.

DayCare Share

Built with Ember.js and Google Maps Javascript API

DaycareShare is a web app that allows users to add and search for daycares. Users can create a user account, login, add a daycare and create a profile for their daycare. Parents can search for nearby daycares that have been added to DaycareShare. Additionally, users can edit their daycare info and add class profiles and teacher profiles to their daycare profile.
